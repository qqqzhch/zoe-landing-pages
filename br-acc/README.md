## 核心价值

### 愿景
构建全球透明度图谱，让数据关系可视化，增强社会信任。

### 痛点解决
- ❌ 复杂的社会/商业关系难以追踪
- ❌ 缺乏统一的透明度评估标准
- ❌ 数据孤岛，信息无法关联

### 我们的解决方案
- ✅ **图谱可视化**: 实体关系一目了然
- ✅ **透明度评估**: 量化指标体系
- ✅ **数据关联**: 打破信息孤岛
- ✅ **开源透明**: 代码公开，结果可信

## 技术栈

- **后端**: Python + FastAPI
- **数据库**: Neo4j (图数据库)
- **前端**: D3.js / Vis.js
- **可视化**: Graphviz / Cytoscape.js
- **部署**: Docker + Kubernetes
