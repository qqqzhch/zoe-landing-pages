## 自动生成流程

这些页面由以下工作流自动生成：

1. 查询 GitHub Trending API
2. 分析商业可行性（Stars、Forks、Issues、技术栈）
3. 生成趋势分析报告
4. 识别高可行性项目（评分 >= 60）
5. 生成完整商业分析
6. 生成 Landing Page（HTML + CSS + README）
7. 部署到本仓库
8. 发布到 Jekyll 博客
